# 清除瀏覽器緩存步驟

## 方法1：完整重新載入（推薦）

1. 打開 `esg-indicators.html` 頁面
2. 按 **Ctrl + Shift + Delete** 打開清除瀏覽器數據
3. 勾選：
   - ✅ 緩存的圖像和文件
   - ✅ Cookie 和其他網站數據
4. 點擊「清除數據」
5. 關閉瀏覽器
6. 重新開啟瀏覽器並載入頁面

---

## 方法2：開發者工具清除

1. 按 **F12** 打開開發者工具
2. 在 Console 輸入並執行：
```javascript
localStorage.clear();
sessionStorage.clear();
location.reload(true);
```

---

## 方法3：無痕模式測試

1. 按 **Ctrl + Shift + N** 開啟無痕模式
2. 直接開啟 `esg-indicators.html`
3. 這樣可以確認是否是緩存問題

---

## 驗證修改是否生效

執行查詢後，在 Console 應該看到：
```
📋 CQL查詢: 抗生素使用率
   CQL來源: Antibiotic_Utilization.cql
   🌐 FHIR伺服器: https://hapi.fhir.org/baseR4
   ✅ MedicationRequest查詢: 500 筆
   📊 結果: 0 抗生素處方 / 500 總處方
   ⚠️ 無抗生素使用資料 (測試伺服器無ATC J01*代碼)
```

然後卡片應該顯示「資料庫無資料」而不是「2」和「0.40%」。
