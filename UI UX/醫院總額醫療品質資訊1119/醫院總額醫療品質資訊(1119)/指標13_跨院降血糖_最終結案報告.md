### 🔧 技術規格

#### 計算公式
```
分子 (Numerator):
  全國跨院同ID不同處方之開始用藥日期與結束用藥日期間
  有重疊之給藥日數 (允許遞延早拿藥)

分母 (Denominator):
  各案件之「給藥日數」總和

重疊率 = (分子 / 分母) × 100%
```

#### 納入條件
1. ✅ 醫院總額之門診案件
2. ✅ 降血糖藥物 (ATC: A10*)
3. ✅ **不分口服及注射** (包含所有劑型)
4. ✅ 排除代辦案件
5. ✅ 同一病人ID
6. ✅ **不同醫院** (跨醫院條件)

#### 重疊天數計算
- 允許提早領藥: 7天
- 計算公式: `GREATEST(LEAST(endDate1+7, endDate2+7) - GREATEST(startDate1, startDate2) + 1, 0)`
