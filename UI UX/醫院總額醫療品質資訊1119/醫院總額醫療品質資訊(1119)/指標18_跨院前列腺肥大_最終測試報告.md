### 計算公式

#### 跨醫院重疊率
```
重疊率 = (跨院重疊日數總和 / 給藥日數總和) × 100%
```

#### 重疊日數計算
- 允許7天遞延早拿藥
- 公式: `GREATEST(LEAST(end_date_a+7, end_date_b+7) - GREATEST(start_date_a, start_date_b) + 1, 0)`

#### 跨院條件
- 關鍵SQL條件: `a.hospital_id != b.hospital_id`
- 同一病患在不同醫院的處方才計入

### FHIR Resource Mapping
- **MedicationRequest** → drug_details (藥品處方資訊)
- **Encounter** → outpatient_claims (門診就醫記錄)
- **Patient** → patient_master (病患主檔)
- **Organization** → hospital_master (醫院主檔)

### 特徵分析
1. **重疊率偏高**: 跨院重疊率在44-52%之間，顯示跨院用藥協調需改善
2. **季度變化**: Q3和Q4重疊率相近，全年平均較高

### 臨床意義
1. **用藥安全風險**: 高重疊率可能導致重複用藥
2. **醫療資源浪費**: 不必要的重複處方
3. **跨院協調需求**: 需加強醫院間資訊共享
