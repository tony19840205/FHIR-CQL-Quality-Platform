## 計算公式比較

### 指標1: 門診注射劑使用率
```
使用率 = (針劑藥品案件數 / 給藥案件數) × 100%

針劑藥品識別:
✓ 醫令代碼長度 = 10碼
✓ 醫令代碼第8碼 = '2'

排除:
✗ 門診化療注射劑 (37005B, 37031B~37041B)
✗ 化療ATC碼 (L01, L02, H01AB01, L03*, L04*, M05*, V10XX03)
✗ 流感疫苗 (J07BB)
✗ 破傷風 (J07AM01)
✗ 急診案件 (case_category='02')
✗ 門診手術 (case_category='03')
✗ 事前審查藥品 (prior_approval_flag='Y')
✗ STAT藥品 (frequency='STAT')
✗ 病人可攜回注射藥品
```

### 指標2: 門診抗生素使用率
```
使用率 = (抗生素案件數 / 給藥案件數) × 100%

抗生素識別:
✓ ATC碼前1碼 = 'J' (抗感染劑)

排除:
✗ 疫苗 (ATC前3碼='J07')
✗ 免疫球蛋白 (ATC前5碼='J06BA')
✗ 急診案件 (case_category='02')
✗ 門診手術 (case_category='03')
✗ 事前審查藥品 (prior_approval_flag='Y')
✗ STAT藥品 (frequency='STAT')
```

## 共同排除條件

兩個指標都排除以下案件:

1. **急診案件** (case_category='02')
2. **門診手術案件** (case_category='03')
3. **事前審查藥品** (prior_approval_flag='Y')
4. **STAT藥品** (frequency='STAT')
5. **代辦案件** (agency_case_flag='Y')

## 給藥案件定義

兩個指標的分母都使用相同定義:

符合以下**任一條件**:
- 藥費 > 0，或
- 給藥天數 > 0，或
- 處方調劑方式 ∈ {1, 0, 6}


## 趨勢分析建議

### 指標1: 注射劑
- 觀察是否有過度使用針劑趨勢
- 比較同層級醫院差異
- 分析口服轉注射比例
- 注意化療藥品排除的正確性

### 指標2: 抗生素
- 監測抗生素抗藥性風險
- 分析不同ATC類別使用趨勢
- 比較口服vs注射途徑比例
- 觀察季節性變化（如流感季）

## 異常值判定標準

### 指標1: 注射劑使用率
- 🟢 正常: 0.5% ~ 1.5%
- 🟡 注意: 1.5% ~ 2.5% 或 < 0.5%
- 🔴 異常: > 2.5%

### 指標2: 抗生素使用率
- 🟢 正常: 15% ~ 25%
- 🟡 注意: 25% ~ 30% 或 < 15%
- 🔴 異常: > 30%

## 重要提醒

### ⚠️ 資料解讀注意事項

1. **基準比較**
   - 應與同層級醫院比較
   - 考慮區域差異
   - 注意季節因素

2. **趨勢觀察**
   - 建議至少觀察4季
   - 環比變化比絕對值重要
   - 突增突減需調查原因

3. **排除條件**
   - 確實套用所有排除規則
   - 化療、急診等排除要正確
   - STAT藥品不應計入

4. **資料品質**
   - 檢查缺失值
   - 驗證代碼格式
   - 確認日期範圍
