## 💊 藥物分類（ATC N05A系列）

### 11個ATC類別

#### 1. **N05AA** - Phenothiazines with aliphatic side-chain
   - 吩噻嗪類 (脂肪族側鏈)

#### 2. **N05AB** - Phenothiazines with piperazine structure
   - 吩噻嗪類 (哌嗪結構)
   - ❌ **排除 N05AB04** (Prochlorperazine)

#### 3. **N05AC** - Phenothiazines with piperidine structure
   - 吩噻嗪類 (哌啶結構)

#### 4. **N05AD** - Butyrophenone derivatives
   - 丁酰苯類衍生物
   - 例如: Haloperidol

#### 5. **N05AE** - Indole derivatives
   - 吲哚衍生物
   - 例如: Molindone, Sertindole

#### 6. **N05AF** - Thioxanthene derivatives
   - 噻噸類衍生物
   - 例如: Flupentixol

#### 7. **N05AG** - Diphenylbutylpiperidine derivatives
   - 二苯丁基哌啶類
   - 例如: Pimozide

#### 8. **N05AH** - Diazepines, oxazepines, thiazepines and oxepines
   - 二氮䓬、噁嗪、噻嗪類
   - 例如: Clozapine, Olanzapine, Quetiapine

#### 9. **N05AL** - Benzamides
   - 苯甲酰胺類
   - 例如: Sulpiride, Amisulpride

#### 10. **N05AN** - Lithium
   - 鋰鹽
   - ❌ **排除 N05AN01** (Lithium carbonate)

#### 11. **N05AX** - Other antipsychotics
   - 其他抗思覺失調症藥物
   - 例如: Risperidone, Aripiprazole

## 🔑 關鍵技術特徵

### 跨醫院邏輯（vs 指標6同醫院）
```sql
-- 指標14（跨醫院）
AND a.hospital_id != b.hospital_id

-- 指標6（同醫院）
AND a.hospital_id = b.hospital_id
```

### 排除條件
1. **N05AB04** (Prochlorperazine) - 主要用於止吐，非典型抗思覺失調症用途
2. **N05AN01** (Lithium carbonate) - 主要用於雙相情緒障礙，非典型抗思覺失調症

### 給藥日數計算
```sql
COALESCE(d.order_drug_day, d.drug_days, 0)
```
優先取醫令檔之 ORDER_DRUG_DAY，若為空則取清單檔之 DRUG_DAYS

### 重疊日數計算（允許遞延早拿藥7天）
```sql
GREATEST(
    DATEDIFF(
        LEAST(DATE_ADD(end_date_a, INTERVAL 7 DAY), 
              DATE_ADD(end_date_b, INTERVAL 7 DAY)),
        GREATEST(start_date_a, start_date_b)
    ) + 1,
    0
)
```

## 🎯 臨床意義

### 跨醫院重疊的影響
1. **醫療資訊共享**: 反映不同醫療院所間的資訊整合程度
2. **病患就醫行為**: 顯示病患在多家醫院求診的情況
3. **藥物安全**: 跨院重複用藥可能導致過量或交互作用風險
4. **醫療資源浪費**: 重複開藥增加健保支出

### 抗思覺失調症藥物特性
- **長期用藥**: 需要持續穩定治療
- **劑量調整**: 需要密切監測副作用
- **交互作用**: 多種藥物併用風險高
- **副作用**: 錐體外徑症狀、代謝症候群等

### 改善建議
1. 加強跨院處方資訊查詢系統
2. 提升病患用藥紀錄可近性
3. 強化藥師跨院用藥諮詢
4. 建立抗思覺失調症藥物專案管理
