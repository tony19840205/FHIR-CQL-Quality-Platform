## 💊 藥物分類（ATC N06A系列）

### 3個ATC類別

#### 1. **N06AA** - Non-selective monoamine reuptake inhibitors
   - 非選擇性單胺再吸收抑制劑
   - 傳統三環抗憂鬱劑 (TCAs)
   - ❌ **排除 N06AA02** (Imipramine - 丙咪嗪)
   - ❌ **排除 N06AA12** (Doxepin - 多塞平)
   - 包含藥物: Amitriptyline, Clomipramine 等

#### 2. **N06AB** - Selective serotonin reuptake inhibitors (SSRIs)
   - 選擇性血清素再吸收抑制劑
   - 最常用的抗憂鬱藥物類別
   - 包含藥物:
     * Fluoxetine (百憂解)
     * Sertraline (樂復得)
     * Paroxetine (克憂果)
     * Citalopram (喜普妙)
     * Escitalopram (立普能)

#### 3. **N06AG** - Monoamine oxidase A inhibitors (MAOIs)
   - 單胺氧化酶A抑制劑
   - 較少使用，需注意飲食限制
   - 包含藥物: Moclobemide 等

## 🔑 關鍵技術特徵

### 跨醫院邏輯（vs 指標7同醫院）
```sql
-- 指標15（跨醫院）
AND a.hospital_id != b.hospital_id

-- 指標7（同醫院）
AND a.hospital_id = b.hospital_id
```

### 排除條件
1. **N06AA02** (Imipramine - 丙咪嗪) - 可能因特殊適應症需求而排除
2. **N06AA12** (Doxepin - 多塞平) - 常用於治療焦慮症及失眠，非典型抗憂鬱用途

### 給藥日數計算
```sql
COALESCE(d.order_drug_day, d.drug_days, 0)
```
優先取醫令檔之 ORDER_DRUG_DAY，若為空則取清單檔之 DRUG_DAYS

### 重疊日數計算（允許遞延早拿藥7天）
```sql
GREATEST(
    DATEDIFF(
        LEAST(DATE_ADD(end_date_a, INTERVAL 7 DAY), 
              DATE_ADD(end_date_b, INTERVAL 7 DAY)),
        GREATEST(start_date_a, start_date_b)
    ) + 1,
    0
)
```

## 🎯 臨床意義

### 跨醫院重疊的影響
1. **醫療資訊共享**: 反映不同醫療院所間的資訊整合程度
2. **病患就醫行為**: 顯示憂鬱症患者在多家醫院求診的情況
3. **藥物安全**: 跨院重複用藥可能導致過量或血清素症候群風險
4. **醫療資源浪費**: 重複開藥增加健保支出

### 抗憂鬱症藥物特性
- **長期用藥**: 需要持續治療4-6個月以上
- **起效緩慢**: SSRIs需2-4週才見效
- **交互作用**: 特別是MAOIs與其他藥物
- **血清素症候群**: 多種血清素藥物併用的風險

### 為何排除 N06AA02 和 N06AA12
- **Imipramine (N06AA02)**: 可能有其他適應症用途
- **Doxepin (N06AA12)**: 常用於焦慮症及失眠治療，非典型抗憂鬱用途

### 改善建議
1. 加強跨院處方資訊查詢系統
2. 提升病患用藥紀錄可近性
3. 強化藥師跨院用藥諮詢
4. 建立抗憂鬱症藥物專案管理
5. 注意血清素症候群風險
