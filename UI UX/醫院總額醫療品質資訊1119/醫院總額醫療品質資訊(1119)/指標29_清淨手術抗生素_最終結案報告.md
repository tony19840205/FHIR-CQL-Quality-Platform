## 📊 指標定義

### 計算公式

```
清淨手術術後使用抗生素超過三日比率 = (分子 / 分母) × 100%

分子：手術後 > 3日使用抗生素案件數
分母：清淨手術案件數
```

### 清淨手術定義

清淨手術為**感染風險最低**的手術類型，須符合以下條件：

**基本條件：**
- 住院西醫院
- 案件分類為5日
- 排除特定診斷（詳見排除條件）

**清淨手術三大類型：**

#### 類型1：下肢關節修復手術（Joint Repair）
- **ICD-10-PCS代碼**（18個）：
  - 膝關節修復：0YQ50ZZ, 0YQ53ZZ, 0YQ54ZZ, 0YQ60ZZ, 0YQ63ZZ, 0YQ64ZZ
  - 髖關節修復：0YQ70ZZ, 0YQ73ZZ, 0YQ74ZZ, 0YQ80ZZ, 0YQ83ZZ, 0YQ84ZZ
  - 踝關節修復：0YQC0ZZ, 0YQC3ZZ, 0YQC4ZZ
  - 跗骨關節修復：0YQE0ZZ, 0YQE3ZZ, 0YQE4ZZ

- **健保處置代碼**（6個）：
  - 75607C（全膝關節置換術）
  - 75610B（全髖關節置換術）
  - 75613C（髖關節修復術）
  - 75614C（膝關節修復術）
  - 75615C（踝關節修復術）
  - 88029C（關節置換術處置費）

#### 類型2：甲狀腺手術（Thyroid Surgery）
- **ICD-10-PCS代碼**（12個）：
  - 甲狀腺切除：0GBG0ZZ, 0GBG3ZZ, 0GBG4ZZ
  - 甲狀腺右葉切除：0GBH0ZZ, 0GBH3ZZ, 0GBH4ZZ
  - 甲狀腺全切除：0GTG0ZZ, 0GTG4ZZ
  - 甲狀腺右葉全切除：0GTH0ZZ, 0GTH4ZZ
  - 甲狀腺左葉全切除：0GTK0ZZ, 0GTK4ZZ

- **健保處置代碼**（4個）：
  - 64162B（甲狀腺全切除術）
  - 64164B（甲狀腺次全切除術）
  - 64169B（甲狀腺葉切除術）
  - 64170B（甲狀腺峽部切除術）

- **主診斷條件**：須有甲狀腺相關診斷
  - ICD-10-CM前3碼：E00-E07, E35
  - ICD-10-CM全碼：E89.0, D44.0

#### 類型3：關節置換術（Joint Replacement）
- **ICD-10-PCS代碼**（164個）：
  - 髖關節置換（0SR9系列、0SRB系列）
  - 膝關節置換（0SRC系列、0SRD系列）
  - 其他關節置換（0SRE-0SRW系列）
  - 詳見CQL完整列表

- **健保處置代碼**（4個）：
  - 64162B, 64164B, 64169B, 64170B

### 排除條件（五大類）

| 排除類別 | ICD-10-CM代碼 | 說明 |
|---------|--------------|------|
| **惡性骨腫瘤** | C40, C41（前3碼） | 骨與關節軟骨惡性腫瘤 |
| **凝血異常** | D65-D68（前3碼） | 凝血功能障礙 |
| **中耳疾病** | H65-H69（前3碼） | 中耳及乳突疾病 |
| **肺炎** | J12-J18（前3碼） | 各類肺炎 |
| **泌尿道感染** | N30, N34（前3碼）, N390（全碼） | 膀胱炎、尿道炎、UTI |

### 抗生素定義

- **ATC代碼**：J01（ANTIBACTERIALS FOR SYSTEMIC USE）
- **使用天數閾值**：> 3日
- **計算方式**：術後首次用藥日至最後用藥日的天數

| `LOINC` | 觀察值代碼（抗生素治療、手術日期） | 4 |
| `http://terminology.hl7.org/CodeSystem/v3-ActCode` | HL7標準代碼 | - |

**核心查詢定義（主要功能）：**

1. **基礎集合定義：**
   - `all_inpatient_encounters`：所有住院案件
   - `quarters`：季度時間定義（8個季度）
   - `annual_period`：年度期間定義

2. **清淨手術判定（分母）：**
   - `joint_repair_procedures`：下肢關節修復手術（類型1）
   - `thyroid_surgery_procedures`：甲狀腺手術（類型2）
   - `joint_replacement_procedures`：關節置換術（類型3）
   - `all_clean_surgery_cases`：所有清淨手術案件（聯集-排除）

3. **排除條件判定：**
   - `has_malignant_bone_tumor`：骨惡性腫瘤
   - `has_coagulation_disorder`：凝血異常
   - `has_otitis_media`：中耳疾病
   - `has_pneumonia`：肺炎
   - `has_urinary_tract_infection`：泌尿道感染

4. **抗生素使用判定：**
   - `postoperative_antibiotic_use`：術後抗生素使用情況
   - `antibiotic_over_3_days_cases`：術後抗生素使用>3日案件

5. **分母計算：**
   - `denominator_clean_surgery_cases`：清淨手術案件
   - `denominator_count`：清淨手術總數
   - `denominator_by_quarter`：按季度分母統計
   - `denominator_annual`：年度分母統計

6. **分子計算：**
   - `numerator_antibiotic_over_3_days`：抗生素>3日案件
   - `numerator_count`：抗生素>3日總數
   - `numerator_by_quarter`：按季度分子統計
   - `numerator_annual`：年度分子統計

7. **指標計算：**
   - `indicator_by_quarter`：季度指標比率
   - `indicator_annual`：年度指標比率
   - `all_periods_results`：所有期間綜合結果

8. **進階分析：**
   - `clean_surgery_type_distribution`：手術類型分布
   - `antibiotic_duration_distribution`：抗生素使用天數分布
   - `hospital_antibiotic_over_3_days_ranking`：醫院別比率排名

**輸出定義（6個）：**
- `Main Output`：主要指標結果
- `Detailed Clean Surgery List`：詳細清淨手術清單
- `Detailed Antibiotic Use`：詳細抗生素使用情況
- `Surgery Type Distribution`：手術類型分布
- `Antibiotic Duration Distribution`：抗生素天數分布
- `Hospital Ranking`：醫院排名
