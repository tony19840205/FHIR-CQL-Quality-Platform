## 📊 指標定義

### 計算公式

```
剖腹產率-初次具適應症 = (分子 / 分母) × 100%

分子: 具適應症之剖腹產案件(不具適應症之剖腹產之外的剖腹產案件)
分母: 生產案件數(自然產案件 + 剖腹產案件)
```

### 分子判定條件（採用排除法）

**具適應症之剖腹產案件 = 所有剖腹產案件 - 不具適應症剖腹產案件**

**不具適應症剖腹產（排除項）** - 符合任一條件：

1. **TW-DRG前三碼為513**
   - 專用於自行要求剖腹產案件

2. **DRG_CODE為0373B**
   - 不具適應症之剖腹產明細代碼

3. **醫令代碼為97014C**
   - 不具適應症之剖腹產處置費

### 分母判定條件

生產案件數（自然產案件 + 剖腹產案件），判定標準與指標26、27相同
