## 🎯 指標定義

### 分子 (Numerator)
同一人(ID) + 同一天(visit_date) + 同一疾病(主診斷ICD-10前三碼) + 同一分區 + 同一院所  
→ 就診 **≥2次** 的人數

**範例:** 
- 病患 `3382b723-dd4f-4b5c-a120-020ba1396659`
- 日期: 2017-10-01
- 醫院: `47b49f1d-ad07-3c2a-a0e2-717b22fe6d15`
- 診斷: K21 (胃食道逆流疾病)
- **就診次數: 5次** ✓ 符合條件

### 分母 (Denominator)
按ID歸戶的門診總人數 (排除診察費=0)

### 排除條件
- ✗ 診察費 = 0
- ✗ 代辦案件
- ✗ 急診案件 (案件分類代碼 02)
- ✗ 門診手術案件 (案件分類代碼 03)
- ✗ 重大傷病病人
- ✗ 慢性精神復健方案
- ✗ 醫療給付改善方案及試辦計畫

### FHIR伺服器
- **名稱:** HAPI FHIR R4 Public Test Server

- **版本:** FHIR R4
- **資料量:** 每次查詢返回最多1000筆

### 查詢參數
```
Encounter?date=ge{StartDate}&date=le{EndDate}&class=AMB&_count=1000
```

- **診斷系統:** ICD-10-CM
- **匹配邏輯:** 精確匹配 (PatientID + Date + HospitalID + Diagnosis3Digit)
