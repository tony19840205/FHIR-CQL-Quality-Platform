## Validation Criteria Applied

### Medication Case Criteria (Denominator)
- Drug cost ≠ 0, OR
- Drug days ≠ 0, OR
- Dispense method in (0, 1, 6)

### Drug Item Count Criteria (Numerator)
- Order type = 1 or 4 (medication order/supply)
- Order code = 10 digits
- Order quantity > 0
- COUNT(DISTINCT drug_code) ≥ 10

### Exclusion Criteria
- Excluded agency cases (agency_flag = 'Y')
- Excluded report reason = '2' cases
