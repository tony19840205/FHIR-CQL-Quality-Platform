## 🔧 技術特色說明

### 1. WHO ATC 編碼系統
所有藥物使用標準 WHO ATC 編碼（`http://www.whocc.no/atc`），符合您的 CQL 指標要求：
- C02-C09：心血管系統藥物（降血壓）
- C10：降血脂藥物
- A10：降血糖藥物
- J01：全身性抗生素

### 2. 時間重疊設計
- **同院重複用藥**：同一次就診開立多種同類藥物
- **跨院重複用藥**：不同醫院在時間重疊期間開立同類藥物（展示專利技術）

### 3. 臨床數值設定
- **血壓**：152/95 mmHg（超過 140/90 標準，判定為控制不良）
- **HbA1c**：9.5%（超過 9.0% 標準，判定為控制不良）
